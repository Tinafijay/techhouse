⌨️ Keyboard shortcuts & help
  • S – Set start marker
  • E – Set end marker
  • Backspace – Apply trim/cut (on export)
  • Ctrl+O – Add logo (image overlay, 5s default)
  • Ctrl+I – Add image overlay (2s default)
  • Ctrl+F – Add sound effect (SFX) at current time
  • Ctrl+M – Add background music
  • Ctrl+X – Render / export video
  • Trim mode: keep selected segment; Cut mode: remove selected segment.
  • After adding an overlay or SFX, use the settings panel to adjust timing, position, etc.
